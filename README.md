# 🍽️ Mess Booking System

## 📌 Overview
This is a Django-based web application where users can register, log in, view meals, and book them online.

---

## 🎯 Features
- User Registration & Login
- View meals (Breakfast, Lunch, Dinner)
- Book meals
- View booking history
- Admin can manage meals and bookings

---

## 🧱 Tech Stack
- Backend: Django (Python)
- Frontend: HTML, CSS
- Database: SQLite

---

## ⚙️ Setup

```bash
pip install django pillow
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
